### Обо мне

Frontend-разработчик с опытом создания сложных веб-программ и мобильных приложений. Специализируюсь на оптимизации производительности, архитектуре и управлении командами.

- **Ключевая экспертиза:** React, TypeScript, Node.js, архитектура SPA
- **Помогаю командам:** расти, внедрять KPI, оптимизировать процессы
- **Спрашивай про:** рефакторинг, code splitting, CI/CD, управление разработкой

## Ключевые достижения

### Оптимизация и производительность
- **SPA для поиска накладок в расписании** — автоматизировал 100% ручных проверок, **экономия 15 рабочих часов в неделю** для администратора
- **Система формирования годового расписания** — сократил время составления с **2 недель до 30 секунд**
- **Интерактивная SPA-карта города Троицк** для Администрации города
- **Мобильное приложение и сайт проекта "Тропа нартов"** — создание и вывод на рыннок
- **Веб-сайт "Центр ткани"** — рефакторинг и оптимизация процессов эксплуатации

### Собственные разработки
- **Веб-сайт ETog** — платформа для поиска и создания мероприятий. Находите свои события по интересам или организовывайте свои собственные
- **Веб-сайт WOOD SHOP** — создание и проектирование дизайна
- **Веб-сайт Zinziver** — разработка под ключ (дизайн + верстка + хостинг)
- **Тестовые задания на позиции** — выполнения тестовых заданий на позиции

### Управление командой
- Координировал работу команды из **5 разработчиков** (Frontend, Backend, QA)
- Внедрил информационную систему анализа KPI → **повысил эффективность команды на 20%**
